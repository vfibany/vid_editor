# 🎬 Local Video Editor

[![Python](https://img.shields.io/badge/Python-3.12.7%2B-blue.svg?logo=python&logoColor=white)](https://www.python.org/)
[![React](https://img.shields.io/badge/React-19.2.7%2B-61DAFB.svg?logo=react&logoColor=black)](https://react.dev/)

A fully local, private video editor and AI voice cloning studio. Edit video timelines and generate high-fidelity, zero-shot personal voice clone narration completely on your own hardware. Edit to your liking.


## 🚀 Quick Start

```bash
cd server/python
python -m venv .venv
pip install
deactivate

cd ../../
pnpm install
pnpm dev
```
